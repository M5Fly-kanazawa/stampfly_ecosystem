# sf version

> **Note:** [English version follows after the Japanese section.](#english) / 日本語の後に英語版があります。

## 1. 概要

sf CLI およびエコシステムのバージョン情報を表示します。

## 2. 構文

```bash
sf version
```

## 3. 出力例

```
StampFly CLI v0.1.0
  Project: stampfly-ecosystem
  Python: 3.12.0
  ESP-IDF: v5.5.2
  Path: /path/to/stampfly-ecosystem
```
